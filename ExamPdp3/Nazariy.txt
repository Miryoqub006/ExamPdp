1-Genericlar 

Genericlar Object keltirib chiqadigan muammolarni hal qilish uchun o'ylab topilgan. Bu degani oldin object bilan ishlanganda quyidagi muammolar bo'lgan bular boxing unboxing , type safetylar . Boxing unboxing muammosi shu ediki value typelardan referencega unboxing esa shuni teskarisi edi bu juda ko'p vaqt olgan va type safetylar ham har tiplar berilishi mumkinligi hisobiga bu xavfli bo'lgan. Mana shu muammolarni Generic hal qiladi. Bu .Netning yuragi bo'lgan CLR ichidagi JIt compiler Ichida amalgam oshadi ya'ni Generic (T) qanaqa type bolishini dastur runtime paytida aniqlab oladi va bu ishni JIT compiler bajaradi. Genericlarni biz classlar field propertylar va methodlarga berishimiz mumkin, class generic qilib ochganimizda bu generic qanaqa type olishi shartligini ham WHERE kalit so'zi orqali bersak bo'ladi (class Test<T> where :) bu yerda class bersak bu degani bizning generic type va reference type bo'lishi kerak agar Struck bersak Value type bo'lishi kerak yana qoshimcha biz classni ozini ham berishimiz mumkin faqat osha class Father class bolishi kerak va generic osha father classning bolalarini type sifatida ishlatib ketadi.
